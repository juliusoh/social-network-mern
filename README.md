# Social Network App
This is a MERN stack application.  It is a small social network app that includes authentication, profiles and forum posts.

## Live Demo
[Link to Live Site](https://lets-connect.juliusoh.com/)

## Features
1. Users can create user login
2. Users can create profile page
3. Users can post a post
4. Users can comment on a post
5. User can like or dislike a post
6. Users can edit profile page through dashboard
7. User authentication via middleware

## Technologies Used
* React.js
* React Router
* Material UI
* Node.js
* Express
* MongoDB
* HTML5
* CSS3
* AWS EC2

## Getting Started
#### 1. Clone the repository and navigate to the directory

#### 2. Install all dependencies 
```shell
npm install
```

#### 3. Compile project
```shell
npm run
```
